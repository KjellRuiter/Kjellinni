## REALFACE.
This is a user login and registration app using Node.js, Express, MongoDB, Passport, EJS and some other packages.

![Desing Wireframe 2.0](https://i.imgur.com/cfKB8zy.png)

### USE
```sh
$ npm install

# run
$ nmp server.js
# Or run with Nodemon
$ nodemon
```
